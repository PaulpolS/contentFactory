# สร้าง Prompt รูป Info (Thai Image Prompt Writer)

สกิลนี้จำลองความสามารถของฟีเจอร์ **"✨ Image Prompt Builder"** ในระบบคลังคำสั่งสร้างรูปและโพส (Prompt Generation Portal) เพื่อใช้คิดหัวข้อคอนเทนต์และเขียน prompt สร้างภาพให้เพจโซเชียลไทย โดยแบ่งงานเป็น 2 ขั้นตอนหลัก: **(A) ออกแบบสไตล์ภาพจากธีมเพจ** และ **(B) เขียน Master Prompt จากหัวข้อ**

อ้างอิงโค้ดต้นฉบับ: `frontend/src/components/PromptGeneratorPortal.tsx` (ฟังก์ชัน `suggestStyles()` และ `generateImagePrompts()`)

## เมื่อไหร่ควรใช้สกิลนี้

ใช้เมื่อผู้ใช้:
- ส่งตัวอย่างคอนเทนต์ที่เคยทำ หรือบอกรายละเอียด/ธีมของเพจมา แล้วอยากได้สไตล์ภาพที่เข้ากับเพจ
- อยากได้ "หัวข้อโพส" ใหม่ ๆ ตามธีมเพจ
- อยากได้ prompt สำหรับ generate ภาพด้วย Google Flow, Imagen 4 หรือ Midjourney จากหัวข้อที่มีอยู่แล้ว

## ขั้นตอนที่ A — วิเคราะห์เพจแล้วออกแบบกลุ่มสไตล์ภาพ (Style Pool)

รับ **รายละเอียดเพจ (Profile Niche)** จากผู้ใช้ (คำอธิบายสั้น ๆ หรือตัวอย่างคอนเทนต์จริง) แล้วทำตามลำดับนี้:

1. วิเคราะห์ "ธีม / อารมณ์ / กลุ่มเป้าหมาย" ของเพจก่อนเสมอ ก่อนเริ่มออกแบบสไตล์
2. ออกแบบสไตล์ภาพ **7 กลุ่ม** โดยทุกกลุ่มต้อง "เข้ากับธีมและอารมณ์ของเพจจริง ๆ" เช่น เพจธรรมะ/สายมู ต้องสงบ ขลัง อบอุ่น มีมิติทางจิตวิญญาณ
3. ภายใต้ธีมเดียวกัน ให้ทำให้ "หลากหลายแบบกลมกลืน" — ต่างกันที่โทนสี เทคนิค มุมมอง องค์ประกอบ เพื่อสลับใช้ให้ฟีดไม่น่าเบื่อ
4. ห้ามใส่สไตล์ที่ขัดกับธีมเพจเด็ดขาด (เช่น เพจธรรมะ ห้ามมี cyberpunk, sci-fi, แนวล้ำอนาคต)

แต่ละสไตล์ในกลุ่มต้องมีข้อมูลครบตามโครงสร้างนี้ (เทียบเท่า `StyleGroup` ในระบบเดิม):

| ฟิลด์ | ความหมาย |
|---|---|
| `name` | ชื่อสไตล์เป็นภาษาไทย เช่น "ขลังทองดวง", "ธรรมะอ่อนนวล" |
| `emoji` | อิโมจิ 1 ตัวแทนสไตล์ |
| `description` | อธิบายสไตล์ 1-2 ประโยค |
| `keywords` | keyword ภาษาอังกฤษสำหรับใช้เขียน prompt (โทนสี, เทคนิค, แสง, องค์ประกอบ) |
| `negativeKeywords` | สิ่งที่ต้องหลีกเลี่ยงในสไตล์นี้ |
| `exampleTopic` | ตัวอย่างหัวข้อโพสที่เข้ากับสไตล์นี้ |

ผลลัพธ์ให้แสดงเป็นตารางหรือ list ที่อ่านง่าย (ไม่จำเป็นต้องเป็น JSON เว้นแต่ผู้ใช้ขอ) เพื่อให้ผู้ใช้เลือกสไตล์ที่จะใช้ต่อในขั้นตอน B

## ขั้นตอนที่ B — เขียน Master Prompt จากหัวข้อ + สไตล์ที่เลือก

รับอินพุตต่อไปนี้จากผู้ใช้ (หรือถามถ้าขาด):
- **หัวข้อโพส** (อาจมีหลายหัวข้อ ทีละบรรทัด)
- **สไตล์ภาพ** ที่เลือกจากขั้นตอน A (ชื่อ, description, keywords)
- **โหมดเนื้อหา**: `infographic` (อินโฟกราฟิก มีข้อความในภาพ) หรือ `general` (ภาพทั่วไป ไม่มีข้อความ)
- **ปลายทาง**: `imagen4`/Google Flow (เขียน prompt เป็นภาษาไทย) หรือ `midjourney` (เขียน prompt เป็นภาษาอังกฤษ)
- **สัดส่วนภาพ** (ถ้ามี): 1:1, 4:5, 9:16, 16:9, 3:4
- **ข้อกำหนดเพิ่มเติม** (ถ้ามี)

จากนั้นเลือกใช้สูตรการเขียนที่ตรงกับ (โหมดเนื้อหา × ปลายทาง) ตาม 4 แบบด้านล่าง — เป็นสูตรเดียวกับที่ระบบเดิมใช้ ให้ยึดตามนี้ **แบบเคร่งครัด**

### B1. Infographic + Imagen 4 / Google Flow (ภาษาไทย) — "Master Prompt"

นี่คือโหมดหลักที่ผู้ใช้เรียกว่า "master prompt" ให้ยึดกฎเหล็ก 5 ข้อนี้เสมอ ห้ามละเมิด:

1. **No Specific Fonts** — ห้ามระบุชื่อฟอนต์เด็ดขาด ให้ใช้ "ข้อความภาษาไทย ตัวหนา อ่านง่าย เด่นชัด" เสมอ
2. **Explicit Text Injection** — ดึงข้อความจริงที่ต้องปรากฏในภาพมาใส่ตรง ๆ ในรูปแบบ: พาดหัวว่า "ข้อความจริง" (ใส่เครื่องหมายคำพูดเสมอ) ห้ามให้ตัวเองแต่งข้อความเอง ต้องใช้ข้อความที่ผู้ใช้กำหนด/หัวข้อที่ให้มา
3. **Structure Breakdown** — ถ้าหัวข้อมีหลายส่วน (เช่น 4 ราศี, 5 ข้อ) ให้แตกเป็นข้อ ๆ ชัดเจน (1. 2. 3. 4.) ระบุเนื้อหาแต่ละช่อง/แต่ละไอคอนแยกกัน ห้ามเขียนรวมเป็นก้อนเดียว
4. **No Page Credits** — ลงท้าย prompt ด้วยประโยคนี้เสมอ (คำต่อคำ):
   > "สำคัญ: ห้ามใส่โลโก้, เครดิตเพจ, ชื่อเว็บ, หรือลายน้ำใดๆ ทั้งสิ้น ข้อความทั้งหมดในภาพต้องเป็นภาษาไทยและสะกดถูกต้องตามนี้"
5. **Output Only** — ส่งเฉพาะตัว prompt เท่านั้น ห้ามเกริ่นนำ ห้ามอธิบาย ห้ามพูดว่า "นี่คือ prompt" หรือ "ต่อไปนี้คือ"

โครงสร้างที่ควรเขียน: บรรยายฉาก/องค์ประกอบหลักของภาพ → ระบุขนาด/สัดส่วน → โทนสีและบรรยากาศตามสไตล์ที่เลือก → พาดหัวหลัก (quote ข้อความจริง) → แตกรายละเอียดแต่ละส่วนเป็นข้อ ๆ พร้อมไอคอน/องค์ประกอบและข้อความในแต่ละช่อง → ปิดท้ายด้วยประโยค "ห้ามใส่โลโก้..." ข้างต้น

**ตัวอย่าง (few-shot ต้นแบบจากระบบเดิม):**

หัวข้อ: `"4 ราศี ดวงการเงินพุ่งแรง เตรียมรับทรัพย์ก้อนโต! (ช่วงนี้)"`
สไตล์: ขลัง มีพลัง โทนทองและน้ำเงินเข้ม จักรราศี

Output:
> สร้างรูปอินโฟกราฟิก ขนาด 1080x1080 ธีมโหราศาสตร์ การเงิน โทนสีทองและน้ำเงินเข้มดูขลัง มีเหรียญทองและถุงเงินลอยอยู่รอบๆ พื้นหลังเป็นวงล้อจักรราศี หัวข้อหลักด้านบนใช้ข้อความภาษาไทย ตัวหนา อ่านง่าย พาดหัวว่า "4 ราศี ดวงการเงินพุ่งแรง เตรียมรับทรัพย์ก้อนโต! (ช่วงนี้)". ด้านล่างแบ่งเป็น 4 ช่องสำหรับ 4 ราศี คือ สิงห์, กันย์, พิจิก, มีน แต่ละช่องมีไอคอนราศีและกองเงิน พร้อมข้อความภาษาไทยสั้นๆ ที่กำหนดไว้ดังนี้: 1. ช่องสิงห์: ไอคอนสิงโตทองคำ ข้อความภาษาไทย: "มีโชคลาภลอยแบบไม่คาดฝัน" 2. ช่องกันย์: ไอคอนผู้หญิงถือรวงข้าวทองคำ ข้อความภาษาไทย: "งานโปรเจกต์ใหญ่สำเร็จ ได้โบนัสก้อนโต" 3. ช่องพิจิก: ไอคอนแมงป่องทองคำ ข้อความภาษาไทย: "การลงทุนได้ผลตอบแทนสูง กำไรงาม" 4. ช่องมีน: ไอคอนปลาคู่ทองคำ ข้อความภาษาไทย: "มีโอกาสได้มรดก หรือผู้ใหญ่เอ็นดูให้ทรัพย์" สำคัญ: ห้ามใส่โลโก้, เครดิตเพจ, ชื่อเว็บ, หรือลายน้ำใดๆ ทั้งสิ้น ข้อความทั้งหมดในภาพต้องเป็นภาษาไทยและสะกดถูกต้องตามนี้

### B2. Infographic + Midjourney

เนื่องจาก Midjourney ไม่วาดข้อความไทยได้ดี ให้แยกเนื้อหากับ prompt ภาพออกจากกัน ตอบ 3 ส่วนตามรูปแบบนี้เท่านั้น:

```
CONTENT:
[เนื้อหาอินโฟกราฟิกภาษาไทย — มีพาดหัว, ประเด็นย่อย 3-5 ข้อ, CTA]

PROMPT:
[Midjourney English visual design prompt — 60-100 คำ อธิบาย layout/สี/ไอคอน/mood เท่านั้น ห้ามใส่เนื้อหาข้อความจริงในนี้]

NEGATIVE:
[negative prompt keywords]
```

### B3. General Image + Imagen 4 / Google Flow (ภาษาไทย บรรยาย)

เขียนเป็น **พารากราฟภาษาไทยที่ลื่นไหล** (natural language ไม่ใช่ keyword list) ความยาว 120-180 คำ ครอบคลุม: สิ่งหลักในภาพ, ฉากหลัง, แสง, บรรยากาศ, สไตล์, โทนสี, มุมกล้อง, คุณภาพของภาพ ตอบเฉพาะตัว prompt เท่านั้น ไม่มีคำนำ

### B4. General Image + Midjourney (English keyword prompt)

ตอบ 2 บรรทัดเท่านั้นตามรูปแบบนี้:

```
PROMPT: [60-120 คำ comma-separated English keywords ปิดท้ายด้วย: masterpiece, best quality, 8k]
NEGATIVE: [negative keywords]
```

## หลักการเลือก input ที่ขาด

- ถ้าผู้ใช้ไม่บอกโหมด/ปลายทาง ให้ถือ default เป็น **infographic + Imagen 4/Google Flow (ภาษาไทย)** เพราะเป็นโหมดหลักที่ระบบเดิมตั้งใจให้เรียกว่า "master prompt"
- ถ้าผู้ใช้ให้ตัวอย่างคอนเทนต์เก่ามาแทนที่จะบอกธีมตรง ๆ ให้สรุปธีม/โทน/กลุ่มเป้าหมายจากตัวอย่างนั้นก่อน แล้วค่อยทำขั้นตอน A
- ถ้าผู้ใช้ให้แค่หัวข้อโดยไม่มีสไตล์ที่เลือกไว้ก่อน ให้เลือกสไตล์ที่เหมาะสมที่สุดเองจากธีมเพจ (หรือถามสั้น ๆ ถ้าไม่แน่ใจ) แล้วระบุในคำตอบว่าเลือกสไตล์ไหนและทำไม
- รักษาความสม่ำเสมอ: หัวข้อเดียวกัน + สไตล์เดียวกัน ควรได้ prompt ที่มีโครงสร้างเดียวกันทุกครั้ง (ต่างกันแค่เนื้อหาตามหัวข้อ)

## สิ่งที่ต้องคิดเพิ่มนอกเหนือจากระบบเดิม (สำหรับ Claude Cowork)

ระบบเดิมต้องให้ผู้ใช้พิมพ์หัวข้อเอง แต่เมื่อใช้กับ Claude Cowork ผู้ใช้จะส่ง "ตัวอย่างคอนเทนต์" มาแทน ดังนั้นให้เพิ่มขั้นตอนก่อน B คือ:

**ขั้นตอนที่ 0 — คิดหัวข้อโพสใหม่**: จากตัวอย่างคอนเทนต์/ธีมเพจที่ผู้ใช้ส่งมา ให้เสนอหัวข้อโพสใหม่ (headline) ที่มีสไตล์การเขียนแบบเดียวกับตัวอย่าง (โทนภาษา, ความยาว, ลูกเล่น เช่น ใช้เครื่องหมายอัศเจรีย์, วงเล็บบอกช่วงเวลา, ตัวเลขนำ) ก่อนจะนำหัวข้อนั้นไปเข้าสูตร B เพื่อเขียน master prompt ต่อ
